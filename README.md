#Tic Tac Toe in Go

###To Run:

Install Go
cd to `tictactoe` directory

1. go run main.go
2. go build, ./tictactoe

###To Test

1. cd `ttt_test`, run `ginkgo`
2. ./test_runner

###Game Rules:
    Simply choose board spot if using Human Player:
        1. 1-9 for 3x3 board
        2. 1-16 for 4x4 board
        3  1-25 for 4x4 board
